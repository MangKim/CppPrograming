1.객체의 생성과 소멸의 의미를 설명하라
```
생성 : 객체가 메모리 공간을 할당받고, 필드(데이터)가 초기화되며 사용 가능한 상태가 되는 과정
```
```
소멸 : 객체가 사용을 마치고 점유하고 있던 메모리 공간을 시스템에 반납하는 과정
```
2.지역변수(지역객체), 전역변수(전역객체)의 차이를 조사하라, 선언위치, 접근범위, 생성시점, 소멸시점, 초기화 여부 등의 관점에서 비교하라
```
크게 지역변수는 한번 쓰고 버리는 변수이고 전역변수는 어디서든 사용할 수 있게끔 만들어 놓은 변수이다.
```
```
선언 위치는 지역변수는 함수 또는 블록 내부, 전역변수는 함수외부 즉 파일상단 등에 선언한다.
```
```
접근범위는 지역은 선언된 블록 내부에서만 사용가능하고, 전역은 프로그램 어디서든 사용가능하다.
```
```
생성지점은 지역은 변수가 선언되어있는 블록에 진입했을때, 전역은 프로그램 시작시 이다.
```
```
소멸지점은 지역은 선언되어있는 블록에서 이탈 할 때 자동으로 소멸한다, 전역은 프로그램 종료시 소멸한다.
```
```
초기화 여부는 지역은 자동으로 초기화가 되지 않기 때문에 직접 초기화 해 주어야 하고, 전역은 0또는 NULL값으로 자동으로 초기화된
```
